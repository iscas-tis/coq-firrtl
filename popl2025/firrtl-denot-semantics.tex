Since each component $cnt$ in $M$ is declared, $cnt$ has a type, say $type(cnt)$. Note that $type(cnt)$ may involve implicit widths.  
On the other hand, all the widths in $vm$ are explicit. 

Let us define $\ftypes$ as the set of pairs of one of the following forms, 
\begin{itemize}
\item $(\gtype, t)$ such that $t \in \gtypes$, 
%
\item $(\vtype, t[n])$ such that $t$ is a type and $n \in \Nat$, 
%
\item $(\btype, f)$ such that $f$ is a field. 
\end{itemize}

We can construct a function $tmap$ that assigns to each identifier $v \in \range{id}$ \tl{where is this defined?} a type where all the widths are explicit, by utilizing the information in $vm$ to instantiate the implicit widths in $type(cnt)$ for components $cnt$ in $M$. 
More precisely, for each $v \in \range{id}$, $tmap(v) = (ft, ori)$ such that $ft \in \ftypes$ where all widths are explicit and $ori \in \{\duplex, \sink, \source\}$.
%\begin{itemize}
%\item $(ori, ground, t)$ such that $ori$ is a Boolean value denoting the orientation of $v$ and $t \in \gtypes$, 
%
%\item $(ori, vector, t[n])$ such that $ori$ is a Boolean value denoting the orientation of $v$ and $t$ is a type where all widths are explicit, 
%
%\item $(ori, bundle, f)$ such that $ori$ is a Boolean value denoting the orientation of $v$ and $f$ is a field where all widths are explicit. 
%\end{itemize}

In the sequel, we first assume that $tmap$ is already constructed and show how to define the semantics of $M$. Later on, we show how to construct $tmap$ from $M$ and $vm$. 

%Let $G= (vm, ct)$ be a module graph. 
The semantics of a statement $s$ in $M$ is defined by a relation $(vm, ct) \xrightarrow[tmap]{s} (vm', ct')$, where $s$ is seen as a modifier of module graphs. Moreover, the relation can be easily extended to sequences of statements, that is, $(vm, ct) \xrightarrow[tmap]{ss} (vm', ct')$, where $ss$ is a sequence of statements. 
       
%Let $M$ be a {\hifirrtl} module and $G$ be a module graph. We define how $G = (vm, ct)$ \emph{matches} $M$ as follows: We define the relations $(vm, ct) \xrightarrow[tmap]{s} (vm', ct')$, where $s$ is a statement. Then $G$ matches $M$ if $(vm_{emp}, ct_{emp}) \xrightarrow[tmap]{ss} (vm, ct)$, where $ss$ is the statement sequence of $M$.
